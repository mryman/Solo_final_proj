Tracking Pledges with Insertion Sort

There are many different sorting algorithms that make sets of data more readable for analyzing trends and patterns. One such algorithm that works well for relatively small data sets is the Insertion Sort. We can demonstrate this with the example scenario below. Each iteration, insertion sort removes one element from the input data, finds the location it belongs within the sorted list, and inserts it there. It repeats until no input elements remain. One advantage of this algorithm is its stability. As noted by the sequence of donors who made equivalent pledges, the algorithm does not change the relative order of elements with equal keys.
